The Azure Certified Solutions Architect - Expert focuses on designing cost- and performance-optimized solutions. It’s ideal for those with AWS or strong on-premises IT experience.

  • Domain 2 — Design Resilient Architectures (26%): Multi-AZ/Region, ALB + Auto Scaling, RDS/Aurora features, Route 53, CRR.

Aprende defaults de cada servicio, límites, y dónde están los límites de seguridad (cuenta/VPC/SG/bucket/key).

Quick Patterns

  • S3 privado: Gateway Endpoint + bucket policy aws:SourceVpce.
  • EKS least-privilege: IRSA (OIDC) con rol por service account.
  • Serverless privado: Interface Endpoints (Secrets Manager, SSM, STS…).
  • Baja latencia global: Global Accelerator para tráfico no-cacheable.
  • Base de datos HA: RDS Multi-AZ; replicas para lectura; Aurora = failover rápido.

Storage & Data

  • S3: versioning, encryption (SSE-S3/KMS), lifecycle, replication, access points; clases: Standard / IA / One Zone-IA / Intelligent-Tiering / Glacier tiers.
  • EBS: gp3 desacopla perf/tamaño; io2/io2 BX para misión crítica; st1/sc1 para throughput/frío.
  • EFS: regional, NFS, multi-AZ; IA/Archive tiers; miles de montajes concurrentes.
  • DynamoDB: PK + sort key; GSIs/LSIs; DAX; on-demand/provisioned; streams para eventos.
  • RDS/Aurora: Multi-AZ para HA; read replicas; backups; Proxy para pool de conexiones.
