Domain 3 — Deploy and manage Azure compute resources (20–25%)

VMs · VMSS · Images · Disks · Updates · App Service · Containers

Compute fundamentals

Favor automation (Bicep/ARM/CLI), HA (zones/sets/scale sets), and security (Managed Identity, private access to PaaS).

Patterns that score

  • Availability: Use availability zones for SLA; availability sets (fault/update domains) where zones aren’t available.
  • Scale: VM Scale Sets with autoscale metrics (CPU, queue length, custom).
  • Images: Shared Image Gallery for versioned images across regions; capture generalized or specialized images correctly.
  • Disks: OS vs data disks; Premium SSD v2 vs Ultra for high IOPS; enable host caching where appropriate.
  • Identity: assign system/user‑assigned managed identity; remove secrets from code.
  • Updates: Update Manager assessment and deployment schedules with maintenance windows.
Tip: If the app needs inbound TLS termination + WAF, prefer Application Gateway; for Layer‑4 + cross‑zone, use Standard Load Balancer.

Common pitfalls

  • Forgetting proximity placement groups for ultra‑low latency multi‑VM deployments.
  • Under‑sizing temp (ephemeral) OS disks and assuming persistence.
  • Letting VMSS instances drift—use extensions, custom script, or config management.

Study checklist

  • VM sizes, disk SKUs, caching modes; disk encryption (platform‑managed vs CMK/host encryption).
  • App Service plans (Free/Shared/Basic/Standard/Premium); deployment slots; VNet integration.
  • Containers: ACR + AKS basics (node pools, upgrades) vs ACI for serverless containers.

Exam Focus

  • Deploy VMs/VMSS with Bicep/ARM and configure identity, networking, and disks
  • Design HA with zones/sets; configure load balancing and autoscale
  • Manage updates, extensions, and run commands
  • Publish apps on App Service with custom domains and private endpoints

Key Azure services

  • Virtual Machines, VM Scale Sets, Shared Image Gallery
  • Azure Load Balancer, Application Gateway (WAF), Front Door
  • Azure Update Manager, Automation Account (runbooks), Desired State Configuration
  • Azure App Service, Azure Container Registry, AKS/ACI

Hands‑on ideas

  • Build a SIG image and deploy a VMSS across zones with autoscale
  • Enable disk encryption with CMK and compare performance with/without caching
  • Publish a web app on App Service with a private endpoint and custom domain
