Domain 5 — Monitor and maintain Azure resources (10–15%)

Azure Monitor · Logs · Alerts · Backup · ASR · Update Manager · Cost

Operate at scale

Centralize telemetry, automate responses, and protect data. Tune ingestion & retention to control cost.

Patterns that score

  • Diagnostics by policy: use DeployIfNotExists to turn on logs/metrics for critical resources and stream to a central Log Analytics workspace.
  • Actionable alerts: metric/log alerts → Action Groups (email/webhook/ITSM/automation); alert processing rules and dynamic thresholds.
  • Backup: soft delete, immutable vaults, cross‑region restore (CRR) for critical workloads; test restores; item vs vault level protection.
  • ASR: VM replication between regions; recovery plans; planned/unplanned failover; test failover networks.
  • Updates: assessment + deployment rings; maintenance windows; pre/post scripts.
  • Cost: Budgets, Advisor, Reservations/Savings Plans; Log Analytics daily cap/table‑level retention.
Tip: Exam loves “configure at scale”—answer with Policy, ARM/Bicep, and templates over portal clicks.

Common pitfalls

  • Sending high‑volume logs without sampling or table retention policies.
  • Disabling soft delete on Backup vaults or skipping multi‑user authorization for vault operations.
  • Failing ASR test because of DNS or NSG rules not mirrored in DR VNet.

Study checklist

  • Azure Monitor primitives: Metrics vs Logs, KQL basics, Workbooks, Action Groups.
  • Alert processing rules vs suppression; dynamic thresholding; metric dimensions.
  • Backup vault SKUs, policies, CRR, soft delete & immutable settings.
  • ASR replication policies, test failover, re‑protection, and runbooks.

Exam Focus

  • Enable diagnostics, route logs, and write KQL for troubleshooting
  • Create metric/log alerts and action groups; configure processing rules
  • Protect workloads with Backup and Site Recovery; perform restores/failovers
  • Automate updates and track compliance

Key Azure services

  • Azure Monitor, Log Analytics, Application Insights
  • Azure Backup, Recovery Services Vault, Azure Site Recovery
  • Azure Update Manager, Automation Accounts, Logic Apps
  • Cost Management + Advisor

Hands‑on ideas

  • Policy‑deploy diagnostics to a workspace + storage + Event Hub
  • Create log alerts with KQL and route to ITSM via Action Group webhook
  • Enable VM backup with immutable vault and run a test restore
  • Configure ASR between paired regions and run a test failover
