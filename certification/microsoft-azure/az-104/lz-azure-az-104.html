AZ-104 · Microsoft Azure Administrator (Associate)

Implementación, gestión y monitorización de identidad, gobernanza, almacenamiento, cómputo y redes en Azure. En esta página tienes el temario, recursos, el enlace al simulador y la práctica.

1. Identidad y gobernanza — Entra ID (AAD), RBAC, PIM, suscripciones, Azure Policy Identidad
2. Almacenamiento — Cuentas, replicación (LRS/ZRS/GZRS), SAS, lifecycle, Azure Files/NFS Datos
3. Cómputo — VMs, Availability Sets, VM Scale Sets, imágenes, extensiones Compute
4. Redes — VNets, subredes, NSG, UDR, NAT, Peering, VPN/ER, Private Link Redes
5. Seguridad y protección — Defender for Cloud, Key Vault, cifrado, Backup, Site Recovery Seguridad
6. Monitorización — Azure Monitor, Log Analytics, alertas, métricas Ops
7. Coste y gobierno — Tags, budgets/costs, cumplimiento Coste

Objetivos de aprendizaje

  • Gestionar accesos con RBAC, PIM y Policy.
  • Desplegar y operar infra con buenas prácticas de compute, redes y datos.
  • Implantar monitorización, resiliencia (Backup/ASR) y seguridad (Defender/Key Vault).
