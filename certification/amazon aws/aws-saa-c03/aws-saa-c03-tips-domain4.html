Deliver required outcomes at the lowest reasonable cost. Mix pricing models, storage tiers, and automation so you don‚Äôt pay for idle capacity.

Core tasks:

  • Right-size compute and choose optimal pricing (On-Demand, Savings Plans, RIs, Spot).
  • Optimize storage & data transfer (S3 lifecycle/tiers, EFS-IA, PrivateLink, CloudFront).
  • Implement visibility/controls (Budgets, Cost Explorer, tags) and automation.

Patterns that score

  • Use serverless/managed (pay per request) when possible.
  • Blend pricing: steady base on Savings Plans/RIs, bursts on On-Demand, batch on Spot.
  • Lifecycle everything: S3 Intelligent-Tiering/IA/Glacier; EFS-IA; snapshot + recycle EBS.
  • Cut transfer costs: PrivateLink/VPC Endpoints, CloudFront, caching.
  • Watch & alert: Budgets, Cost Anomaly Detection, CUR, Trusted Advisor.

Common pitfalls

  • Oversizing EC2/EBS (paying for unused IOPS/CPU).
  • Keeping cold data on S3 Standard/EFS Standard.
  • Egressing to AWS services over the public internet instead of endpoints.
  • Leaving dev/test running at night or weekends.
  • No tagging ‚Üí no showback/chargeback.

Study checklist

  • Savings Plans vs RIs (standard vs convertible); Spot interruptions & capacity-optimized strategy.
  • S3 classes + lifecycle; Glacier Instant/Flexible/Deep Archive.
  • EBS gp3 tuning (IOPS/throughput), snapshots, EFS-IA.
  • Cost tools: Budgets, Cost Explorer, Anomaly Detection, CUR, Trusted Advisor.
  • Architectural levers: CloudFront caching, PrivateLink, compression, batching.

Exam Focus

  • Choose the cheapest option that still meets perf/RTO/RPO.
  • Turn off idle capacity via Auto Scaling or serverless.
  • Move aged data to cheaper tiers automatically.
  • Prefer edge/local access to reduce transfer and latency.

Key AWS services

  • EC2 + Savings Plans/RIs/Spot, Lambda
  • S3 (all classes), EFS/EFS-IA, EBS, FSx
  • RDS/Aurora (incl. Serverless), DynamoDB On-Demand/Provisioned
  • CloudFront, Global Accelerator, PrivateLink/VPC Endpoints
  • Cost Explorer, Budgets, CUR, Trusted Advisor

Hands-on ideas

  • Enable S3 lifecycle to Glacier and estimate savings.
  • Convert a steady EC2 workload to Savings Plans.
  • Put CloudFront in front of S3 and compare egress.
  • Set Cost Anomaly Detection + Budgets email alerts.
