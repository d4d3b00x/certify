AZ-305 — D1: Identidad, Gobernanza y Monitorización

Claves rápidas

  • Diseña RBAC a mínimo privilegio. Usa PIM para elevaciones temporales.
  • Azure Policy para cumplimiento (denegar, auditar, desplegar configuración).
  • Arquitecturas hub-spoke con Azure Firewall y DNS privado.
  • Monitorización con Azure Monitor, Log Analytics y Activity Log.

Patrones que caen

  • Landing Zone + CAF: políticas, blueprints, etiquetas y gestión de costes.
  • Segregación por suscripciones y grupos de gestión.
